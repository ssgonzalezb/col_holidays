# col_holidays
Proyecto en FastAPI que provee los siguientes servicios:
1. Determina si una fecha dada es festivo
2. Devuelve la fecha de la Pascua
3. Devuelve los festivos de fecha fija de un año determinado
4. Devuelve los festivos de fecha variable (Ley Emiliani) de un año determinado
5. Devuelve los festivos basados en la Pascua de un año determinado
6. Devuelve todos los festivos de un año determinado

Aplica para años posteriores a 1583 y sólo para Colombia, en un próximo feature se irán añadiendo más países.
